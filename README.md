# 📊 Uso de Planilhas Eletrônicas para Carga, Tratamento e Análise de Dados

## 📖 Sobre o Projeto

Este projeto tem como objetivo aplicar técnicas de carga, tratamento e análise de dados utilizando o **Microsoft Excel**, explorando recursos que auxiliam na extração de informações relevantes para suporte à tomada de decisões.

A análise de dados desempenha um papel fundamental em diversas áreas, como meteorologia, política, economia e gestão, tornando essencial o domínio de ferramentas capazes de organizar, transformar e interpretar grandes volumes de informação. Nesse contexto, o Microsoft Excel foi utilizado como plataforma principal devido à sua ampla disponibilidade e versatilidade para manipulação de dados.

O trabalho foi dividido em dois estudos de caso, desenvolvidos a partir de bases de dados públicas e com diferentes níveis de complexidade.

---

## 🗳️ Estudo 1 – Análise do Eleitorado Brasileiro

O primeiro estudo foi realizado utilizando dados do eleitorado brasileiro disponibilizados pelo Tribunal Superior Eleitoral (TSE).

### Fonte dos Dados

* TSE – Eleitorado Atual

### Escopo da Análise

* Estado do Acre (AC)
* Estado do Amapá (AP)

### Técnicas Aplicadas

* Limpeza e organização dos dados
* Aplicação de filtros e ordenação
* Uso de fórmulas estatísticas
* Criação de gráficos para visualização dos resultados
* Análise exploratória do perfil do eleitorado

---

## 🌦️ Estudo 2 – Análise de Dados Meteorológicos

O segundo estudo utilizou dados meteorológicos disponibilizados pelo Instituto Nacional de Meteorologia (INMET).

### Fonte dos Dados

* Instituto Nacional de Meteorologia (INMET)

### Escopo da Análise

* Dados de uma estação meteorológica
* Período analisado: 2023 e 2024

### Técnicas Aplicadas

* Tratamento de grandes volumes de dados
* Tabelas dinâmicas
* Correlação entre variáveis meteorológicas
* Construção de gráficos analíticos
* Identificação de padrões e tendências climáticas

---

## 🎯 Objetivos de Aprendizagem

Durante o desenvolvimento do projeto foram aplicados conceitos relacionados a:

* Carga e preparação de dados;
* Limpeza e validação de informações;
* Análise exploratória de dados (EDA);
* Construção de visualizações gráficas;
* Uso de tabelas dinâmicas;
* Aplicação de funções e fórmulas avançadas do Excel;
* Interpretação e apresentação de resultados.
